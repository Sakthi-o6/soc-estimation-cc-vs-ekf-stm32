# soc-estimation-cc-vs-ekf-stm32
Real-time implementation and comparison of Coulomb Counting and EKF for battery SOC estimation on STM32, analyzing accuracy and CPU usage.
This project implements and compares two SOC estimation methods—Coulomb Counting and Extended Kalman Filter (EKF) on an STM32-based embedded system. Both algorithms run in real time using RTOS, and their performance is evaluated based on estimation accuracy and CPU usage. The goal is to study the trade off between accuracy and computational cost for practical embedded battery management applications.
